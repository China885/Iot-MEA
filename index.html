อุปกรณ์ IoT / เซนเซอร์ / แอคชูเอเตอร์

ESP8266 โมดูล Wi-Fi ราคาเป็นมิตร เหมาะเริ่มทำ IoT เบื้องต้นและงานสั่งงานผ่านเน็ต
มี GPIO พอสำหรับโปรเจ็กต์เล็ก ๆ ใช้กับ Arduino Core ได้สะดวก
ESP32 MCU Wi-Fi + Bluetooth แรงกว่า ESP8266 เหมาะงานหลายงานพร้อมกัน
มี ADC/DAC/Touch/PIO เยอะ ทำ IoT, เซนเซอร์, เสียง ได้ครบ
Raspberry Pi Pico W ชิป RP2040 + Wi-Fi เหมาะทำงาน IoT ที่ต้องการ I/O เยอะและต้นทุนต่ำ
เขียนได้ทั้ง MicroPython/Arduino C++ ประหยัดไฟ ใช้งานยืดหยุ่น
Arduino MKR WiFi 1010 บอร์ด Arduino ขนาดเล็กมี Wi-Fi/BLE ในตัว เหมาะงานเชื่อมคลาวด์
มีวงจรชาร์จ Li-Po บนบอร์ด ออกแบบเพื่อ IoT โดยเฉพาะ
Arduino Nano 33 IoT Nano รุ่น IoT มี Wi-Fi/BLE บอร์ดจิ๋วแต่ครบเครื่อง ใช้งานสะดวกในพื้นที่จำกัด
เหมาะโซลูชันสวมใส่/เซนเซอร์กระจายหลายจุด
DHT22 / AM2302 เซนเซอร์อุณหภูมิ/ความชื้น ดิจิทัล ใช้งานง่าย ราคาคุ้มค่า
ความไวปานกลาง เหมาะงานในอาคาร ไม่ทนสภาพกลางแจ้งจัด
BME280 วัดอุณหภูมิ ความชื้น และความดันอากาศ ในตัวเดียว ความแม่นยำดี
ต่อ I²C/SPI ได้ เหมาะสถานีอากาศขนาดจิ๋ว
BH1750 เซนเซอร์วัดแสงหน่วยลักซ์ ต่อ I²C ค่าเสถียร อ่านตรงไปตรงมา
ใช้ควบคุมไฟอัตโนมัติ/บันทึกสภาพแสง
PIR Motion ตรวจจับการเคลื่อนไหวของคน/สัตว์ด้วยอินฟราเรดแบบพาสซีฟ
กินไฟน้อย เหมาะสัญญาณกันขโมย/เปิดไฟอัตโนมัติ
MQ-2 เซนเซอร์ก๊าซไวต่อควัน/ก๊าซหุงต้ม ราคาถูก ใช้งานกว้างขวาง
ต้องคาลิเบรตและวางในที่อากาศถ่ายเทจึงอ่านสเถียร
รีเลย์โมดูล สวิตช์ไฟฟ้าควบคุมด้วยสัญญาณ เหมาะแยกวงจรโหลดกับไมโครคอนโทรลเลอร์
เลือกแรงดัน/กระแสให้พอ และคำนึงถึงออปโต/ไดโอดกลับขั้ว
Solid State Relay (SSR) สวิตช์อิเล็กทรอนิกส์ไร้หน้าสัมผัส เงียบ สึกหรอน้อย
ร้อนกว่าริเลย์กลไก ควรติดฮีตซิงและใช้ตามพิกัด
มอเตอร์ DC + ไดรเวอร์ ขับง่ายด้วย PWM ควบคุมความเร็ว-ทิศทางด้วย H-Bridge (เช่น L298N)
ใส่ไดโอดกันกลับและเลือกแหล่งจ่ายแยกจากลอจิก
เซอร์โว (PWM) หมุนตามสัญญาณ PWM เป็นองศา เหมาะหุ่นยนต์/กลไกเล็ก
ต้องแหล่งจ่ายนิ่ง กระแสพอ ไม่ดึงจากพอร์ต 5V บอร์ดโดยตรง
สเต็ปเปอร์ หมุนเป็นสเต็ปละเอียด เหมาะงานที่ต้องตำแหน่งแม่น เช่น CNC/ปริ้น 3D
ใช้ไดรเวอร์เฉพาะ (A4988/TMC) และจัดการความร้อนให้ดี

ชิ้นส่วนวงจร (พื้นฐานถึงใช้งานจริง)

ตัวต้านทาน จำกัดกระแส/แบ่งแรงดัน เลือกค่าด้วยรหัสสีและกำลังวัตต์ให้พอ
ใช้ร่วมกับ LED/ไอซีเพื่อปกป้องวงจรไม่ให้เกินพิกัด
ตัวเก็บประจุ กรองสัญญาณ/คัปปลิง/จ่ายฉับพลัน มีหลายชนิด (อิเล็กโทรไลต์/เซรามิก/ฟิล์ม)
ระวังขั้วกับแรงดันทน โดยเฉพาะอิเล็กโทรไลต์
ตัวเหนี่ยวนำ เก็บพลังงานในสนามแม่เหล็ก ใช้กรอง/สวิตชิ่งเพาเวอร์ซัพพลาย
ค่าอิมพีแดนซ์ขึ้นกับความถี่ เลือกกระแสอิ่มตัวให้พอ
ไดโอด ป้องกันกระแสย้อน/ทำเรกติไฟเออร์ ใส่กับคอยล์เพื่อกันแรงดันย้อน
มีชนิดช็อตกี้/ซีเนอร์ เลือกตามงานความถี่-แรงดัน
ทรานซิสเตอร์ (BJT) ขยาย/สวิตช์สัญญาณ เลือกตาม hFE และพิกัด I/V/กำลังสูญเสีย
ต้องใช้ไบอัสและรีซิสเตอร์ที่เหมาะสมเพื่อสวิตช์อิ่มตัว
MOSFET สวิตช์ประสิทธิภาพสูง ควบคุมด้วยแรงดันเกต Rds(on) ต่ำช่วยลดความร้อน
เลือกแบบลอจิกเลเวลเมื่อขับด้วย 3.3/5V จากไมโครคอนโทรลเลอร์
IC / ไมโครคอนโทรลเลอร์ หัวใจของงานดิจิทัล เลือกตามคอร์, ความถี่, RAM/Flash และ I/O
คิดเรื่องแรงดันเลี้ยง, เพาเวอร์, และเครื่องมือพัฒนาให้พร้อม
แหล่งจ่ายไฟ (Buck/Boost) แปลงแรงดันให้เสถียร เลือกกระแสสำรองเผื่อ 20–30%
ใส่คาปาซิเตอร์ใกล้โหลดและออกแบบกราวด์ให้สะอาด
คอนเน็กเตอร์/เทอร์มินัล จุดต่อสาย/บัส เลือกตามกระแสและการยึด เช่น สกรู/สปริง/ดูรองท์
สำหรับงานสั่นสะเทือน ใช้หัวเข้าคลิปล็อกหรือเทอร์มินัลบล็อก
เครื่องมือช่าง (พื้นฐาน) คีมตัด/ปอก/ย้ำสาย, หัวแร้งบัดกรี, มัลติมิเตอร์ — ของจำเป็นทุกโต๊ะ
สำหรับงานไฟฟ้าเลือกคีมมีฉนวนมาตรฐาน VDE/IEC60900 เพื่อความปลอดภัย

การเดินสายไฟบ้าน (สรุปย่อ + ความปลอดภัยมาก่อน)

⚠️ คำเตือน: งานไฟฟ้าควรทำโดยผู้มีใบอนุญาตหรืออยู่ในการกำกับของช่างไฟฟ้าที่มีความรู้ ปฏิบัติตามมาตรฐานการติดตั้งไฟฟ้าประเทศไทย/MEA และตัดไฟก่อนทำงานเสมอ. อ้างอิงแนวทางจากเอกสารของ MEA.
ชนิดสายไฟที่พบบ่อย (บ้าน) VAF/VAF-G เดินลอยในบ้าน แรงดัน 300/500V ห้ามร้อยท่อ/ฝังดิน; NYY/NYY-G เดินภายนอก/ฝังดินได้; THW ใช้เมื่อร้อยท่อ เลือกขนาดหน้าตัดตามกระแสและระยะทาง
สีสาย (แนวทางทั่วไป) โดยแพร่หลาย: สายนิวตรอน มักใช้ ฟ้า, สายดิน เขียว/เหลือง, สายไฟเฟส สีน้ำตาล/ดำ/เทา (ขึ้นกับระบบ/มาตรฐาน). ตรวจสอบมาตรฐานหน้างานเสมอ
อุปกรณ์ป้องกัน ใช้ MCB แยกวงจร และ RCD/RCBO ป้องกันไฟดูด เลือกพิกัดตามโหลดและกราวด์ถูกต้อง
ติดตั้งตู้เมน/สายดิน/หลักดินตามมาตรฐานการติดตั้ง
แนวทางเดินสาย เดินในท่อ/รางให้เรียบร้อย หลีกเลี่ยงโค้งหักงอเกินจำเป็น และแยกสายแรงดันกับสัญญาณ
ยึดระยะคลิป/เข็มขัดรัดสายสม่ำเสมอ มีเสตนซิลระบุวงจรที่ตู้
คีมที่ควรมี คีมตัด (ทั่วไป/ฉนวน/VDE), คีมปอกสายปรับขนาด, คีมย้ำหางปลา/หัวคอนแทค
เลือกชนิดให้ตรงงานและใช้ของมีฉนวนสำหรับงานไฟฟ้าเสมอ
เต้ารับ/สวิตช์ เลือกมาตรฐาน มอก. ติดตั้งกล่องพักและต่อสายให้แน่น ใช้หางปลาย้ำเมื่อจำเป็น
ตรวจสอบทิศ L-N-E ให้ถูกต้องก่อนจ่ายไฟ

Uno R3 บอร์ดคลาสสิก ATmega328P เหมาะเริ่มต้น ชุมชนใหญ่ ไลบรารีเยอะ
พอร์ต/หน่วยความจำจำกัด เหมาะโปรเจ็กต์พื้นฐาน
Uno R4 (Minima/WiFi) MCU 32-บิต เร็วและ RAM/Flash มากกว่า R3; รุ่น WiFi มี Wi-Fi/BLE ในตัว
ไลบรารีส่วนใหญ่รองรับ ย้ายโปรเจ็กต์จาก R3 ได้ไม่ยาก
Mega 2560 I/O เยอะ เหมาะโปรเจ็กต์ต้องใช้พินจำนวนมาก เช่น CNC/3D Printer
ช้ากว่า 32-บิต แต่เสถียรและหาของง่าย
Nano / Nano Every ขนาดจิ๋ว เสียบเบรดบอร์ดได้ เหมาะต้นแบบพื้นที่จำกัด
รุ่น Every อัปเกรดสเปกและความเข้ากันได้ที่ดีขึ้น
Nano 33 BLE / Sense มี BLE และรุ่น Sense มี IMU/ไมค์ เหมาะงาน ML ขอบสนาม (TinyML)
ใช้ 3.3V ระวังต่อวงจรร่วมกับ 5V
Due 32-บิต ARM ประสิทธิภาพสูง เหมาะงานต้องการความเร็ว/พอร์ตมาก
ใช้ลอจิก 3.3V ระวังการต่อร่วมกับอุปกรณ์ 5V
Leonardo / Micro รองรับ HID เป็นคีย์บอร์ด/เมาส์ได้โดยตรง เหมาะงานอินพุตคัสตอม
I/O ไม่มาก แต่ลูกเล่นด้าน USB เด่น
MKR Zero / MKR WiFi 1010 บอร์ดตระกูล MKR สำหรับ IoT/เสียง/การ์ด SD; รุ่น WiFi มีเน็ตในตัว.
ฟอร์มแฟกเตอร์เล็ก พกง่าย ใช้งานกับแบต Li-Po สบาย
Portenta H7 / GIGA R1 / Nicla ระดับโปรฯ: สองคอร์แรง, พอร์ตเยอะ, เซนเซอร์จิ๋วสำหรับงานอุตสาหกรรม/AIoT

อ้างอิงรายการบอร์ดจากหน้ารวมฮาร์ดแวร์/สโตร์ Arduino (ดูแหล่งอ้างอิง)

Arduino IDE (สรุปข้อดี/ข้อจำกัด)

ข้อดี ติดตั้งง่าย ใช้ได้ทุกแพลตฟอร์ม มี Library/Board Manager, Serial Monitor/Plotter, Auto-format/Autocomplete
ชุมชนใหญ่ เอกสารครบ เชื่อม Remote Sketchbook/Arduino Cloud ได้
ข้อจำกัด การดีบักฮาร์ดแวร์ต้องตั้งค่า/อุปกรณ์เพิ่ม (เช่น J-Link/Atmel-ICE)
ฟีเจอร์ขั้นสูง/รีแฟกเตอร์โค้ดสู้ IDE ใหญ่ ๆ ไม่ได้ในบางจุด
เริ่มใช้งานเร็ว ดาวน์โหลด IDE 2 → ติดตั้งบอร์ดจาก Board Manager → เปิดตัวอย่าง → Verify/Upload
ถ้าใช้ Wi-Fi บางรุ่น อัปเดตเฟิร์มแวร์/อัปโหลดใบรับรอง SSL ตามคู่มือ
